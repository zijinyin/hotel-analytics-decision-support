# Data Notes

This folder contains sample inputs and sample outputs used for demonstration and reproducibility.

## Included
- `sample_inputs/`: small example input files for booking and customer workflows
- `sample_outputs/`: example prediction outputs generated by the trained models

## Raw Data
Raw data files may be large and may not always be appropriate for public version control.
If needed, place raw files under `data/raw/` locally when reproducing the project.

## Reproducibility
To rerun the pipeline, update the training and scoring script input paths to match your local data location.
